## Reporting a Vulnerability

If you discover a security vulnerability within Gestore Spese, please send an e-mail to alsampognaro@gmail.com. All security vulnerabilities will be promptly addressed.

Please include the following details in your report:
*   Description of the vulnerability
*   Steps to reproduce the vulnerability
*   Affected component(s)
*   Any potential impact

We appreciate your help in keeping "Gestore Spese" secure!
